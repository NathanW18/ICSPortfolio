2D Arrays

2D arrays are a special type of Java arrays. That is, they can store information in two dimensions. 2D arrays are like a rectangular table. You have an information category on the rows and an information category on the columns. For example, you could store vegetables in a 2D array representing a farmerâ€™s plot, with x and y coordinates being the columns and rows, respectively.

Here is an image of a 2D array.

 

Declaration of a 2D array is quite straightforward and similar to that of a regular 1D array.

int[][] plot = new int[10][10]; // creates a 10 x 10 integer 2D array called plot

We can refer to the image above to access an element of a 2D array: arrayName[rowNumber][columnNumber]. Here, arrayName is the name of the array, rowNumber is the index of the row (begins at 0), and columnNumber is the index of the column. For example, element 2 would be at myArray[0][1].

To loop through a 2D array, we want to loop through the rows and columns. We can do this through a for loop because we know how the lengths of rows and columns.

for (int i = 0; i < myArray.length; i++) {
        for (int j = 0; j < myArray[i].length; j++) {
                int element = myArray[i][j];
                System.out.println(element); // prints the element out
        }
    }

Common mistakes include accidentally swapping the rows and columns when you try to access an element of a 2D array.

Applications

  • What it looks like
    • Farming plot, make, coordinate plane
  • Store information on rows and columns to create a database
    • Ex: Rows could represent days, and columns could represent amount of rain
  • Information that multiple arrays could store (i.e., parallel arrays) all in one large 2D array
